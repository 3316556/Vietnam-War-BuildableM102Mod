# Buildable M102 Howitzer

When the USA player builds the artillery sandbag emplacement, this also places a working M102 howitzer at the emplacement.

## Install

1. Install MelonLoader for the IL2CPP Windows version of *Vietnam War*.
2. Download the release ZIP for this mod.
3. Extract it into your game folder, usually SteamLibrary\steamapps\common\VietnamWar.
4. Confirm that Mods\BuildableM102Mod.dll exists in the game folder.
5. Launch the game.

To uninstall, delete Mods\BuildableM102Mod.dll from the game folder.

## Compatibility

Built for the current local Steam/MelonLoader IL2CPP setup used by the author. Other game versions may need a rebuilt DLL.
